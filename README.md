## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* yarn
  ```sh
  npm i -g yarn
  ```

### Installation


1. Clone the repo
   ```sh
   $ git clone https://github.com/dan-online_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   $ yarn
   ```
4. Start dev environment
   ```sh
   $ yarn dev
   ```
